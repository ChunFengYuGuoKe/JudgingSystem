## 基于 AST 的程序源码分析
四川大学软件学院 2023-2024 学年春，软件项目管理课程报告，详情请阅读项目计划书

### 项目介绍

搭建一个面向教学服务的 web-server 服务器，实现基本的用户增删查改功能，对学生作答的代码进行黑盒测试，以及基于 AST 的白盒测试。

### 主要任务

搭建 web-server 服务器，实现 Java 语言判断分析模块，集成现有的 python 和 shell 语言判断分析模块。
